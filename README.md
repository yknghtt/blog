# Полка — твой блог о фильмах и книгах

## Что это
Простой личный блог: лента записей с фильтром «Фильмы / Книги» и форма добавления записи прямо на сайте.
Все записи хранятся в файле `data/entries.json` в этом же репозитории.

## Как опубликовать (один раз)

1. Зайди на github.com (ты уже зарегистрирована).
2. Создай новый репозиторий: кнопка **New repository**.
   - Имя, например: `blog`
   - Сделай его **Public**
   - Не добавляй README при создании (он у нас уже есть)
3. Загрузи туда все файлы из этой папки (index.html, add.html, settings.html, styles.css, store.js, data/entries.json, README.md) — это можно сделать через кнопку **Add file → Upload files** на странице репозитория, просто перетащив файлы.
4. В репозитории зайди в **Settings → Pages**.
   - В разделе **Source** выбери **Deploy from a branch**
   - Branch: **main**, папка **/ (root)**
   - Нажми **Save**
5. Через минуту-две сайт появится по адресу:
   `https://yknghtt.github.io/blog/`

## Как подключить сохранение записей

1. Открой сайт → **настройки** (ссылка внизу страницы)
2. Получи личный токен на github.com/settings/tokens/new (подробно расписано прямо на странице настроек сайта)
3. Введи имя пользователя, название репозитория и токен — сохрани

После этого форма «+ Новая запись» начнёт реально сохранять записи в репозиторий.

## Важно про токен
Токен даёт доступ только на запись в этот конкретный репозиторий (если ты выбрала ограничение по repository access).
Он хранится только в браузере (localStorage), никуда больше не отправляется.
